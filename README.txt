Description
===========

Policy product for Ploomcake.

NOTE: this is not a standalone package, thus you cannot install it individually.
Please visit http://www.ploomcake.org/en/download to get Ploomcake.

Ploomcake is a Plone distribution that allows you to create a ready-to-use Plone website, collaboration portal or
intranet, already configured with many addons.
